# Week 3 — Algorithms

**Topic**: Sorting algorithms, searching, time complexity, voting systems

## Problems

### Plurality
Simulates a plurality vote election.
Each voter picks one candidate, the one with the most votes wins.

### Runoff
Simulates a ranked-choice runoff election.
Eliminates the last-place candidate each round until someone has a majority.

### Sort
Analyzed three mystery sorting algorithms (sort1, sort2, sort3) by running them on
random, reversed, and sorted inputs and measuring performance.
Identified each as bubble, selection, or merge sort based on behavior.
